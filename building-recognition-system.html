Building Recognition System

Project Overview

This project aims to classify images of buildings into predefined categories using the VGG19 convolutional neural network. It is designed to be highly accurate and efficient, making it suitable for applications in urban planning, real estate analysis, and automated image tagging.

Technologies Used

  • Deep Learning Framework: PyTorch
  • Model Architecture: VGG19
  • Programming Language: Python
  • Libraries: NumPy, Matplotlib, OpenCV
  • Integrated Development Environment: GPU: 1*P100 (16GB) | CPU: 64 GB
