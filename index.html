Fine-tuning common NLP tasks based on Transformers GitHub Link

Background:

This project aims to fine-tune common Natural Language Processing (NLP) tasks based on Transformer models. By utilizing datasets provided by Hugging Face, we constructed various models to address different NLP problems, including question-answering systems, sentence similarity, text classification, token classification, text generation, summarization, and translation. The goal of the project is to enhance the performance of the models on specific tasks while exploring the effectiveness of different fine-tuning strategies.

Analysis of Model Averaging Methods in Survival Analysis GitHub Link

Background:

This project aims to explore the application of model averaging methods in predicting survival time for breast cancer patients. By constructing three survival analysis models (including the Cox proportional hazards model, Weibull model, and accelerated failure time model based on log-normal distribution), we employ three model averaging methods (Bayesian model averaging, Bagging, and weighted model averaging) to enhance the accuracy of predictions. Additionally, we compare the effectiveness of various weight selection methods (such as AIC, BIC, Mallows, and Jackknife) to ensure the chosen models are effective in practical applications.
